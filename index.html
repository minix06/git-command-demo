Add

git add <filename> ajoute un fichier dans l'index

git config --local user.email <user email> Initialise un email d'utilisateur

Commit

git commit -m "your message" commit des fichier de la zone staging vers le repository

git commit -a commit des fichier de la zone staging vers le repository et ouvre l'éditeur par défaut pour editer le message

Affichage log

git log affiche les commits

git log -1 affiche le dernier commit

git log --pretty=oneline --max-count=2 affiche sur une ligne les 2 derniers commits

git log --pretty=oneline --since="5 minutes ago" affiche les derniers commits dans les 5 minutes

git log --pretty=oneline --until="5 minutes ago" affiche les derniers commits dans les 5 minutes

git log --pretty=oneline --author="Muriel M" affiche les commits d'un auteur (case sensitive)

git log --pretty=oneline --author=^"Mur" affiche les commits d'un auteur commence par Mur(case sensitive)

git log --oneline --all --pretty=format:"%h %cd %s (%an) %d" --since="1 days ago" affiche les commits d'un auteur commence par Mur(case sensitive)

format de log

  • --pretty="..." format de sortie
  • %h "abréviation de hash code commit"
  • %d " commit décoration"
  • %ad commit date
  • %s commentaires
  • %an author name
  • --graph commit tree
  • --date=... (du formatage standard)
